Example: Make a box plot

raynamharris/r4rnaseq#3

After tidying, lengthening, and joining 3 data frames, we can make a box plot of differentially expressed genes. Use facet_wrap() to generate a plot for each gene, use theme() to change the font angle and face. Use scale_y_log10(labels = label_number_si()) modify the y axis. 

counts_tidy_long_joined %>%
  ggplot(aes(x = AGE, y = counts)) +
  geom_boxplot() +
  geom_point() +
  facet_wrap(~name, scales = "free_y") +
  theme(axis.text.x = element_text(angle = 45, hjust  = 1),
        strip.text = element_text(face = "italic")) +
  scale_y_log10(labels = label_number_si())
